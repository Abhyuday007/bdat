Week 7 : 
Basic HBase Commands (HBase Shell):
Start Hadoop services:
start-dfs.sh
start-yarn.sh
Next Start HBase with following commands
cd /usr/local/hbase
bin/start-hbase.sh
Now check status again by
jps
(you should able to see HRegionServer, HQuorumPeer, HMaster)Next run hbase shell:
./bin/hbase shell
Creating a Table
create 'students', 'info', 'contact'
Inserting Data
put 'students', '1', 'info:name', 'Alice'
put 'students', '1', 'info:age', '22'put 'students', '1', 'contact:email', 'alice@gmail.com'
Retrieving Data
get 'students', '1'
scan 'students'
Updating Data
put 'students', '1', 'contact:email', 'alice_new@gmail.com'
Deleting Data
delete 'students', '1', 'contact:email'
deleteall 'students', '1'
Disabling a Table
disable ‘tablename’
is_disabled 'table name’
Enabling a Table
enable 'tablename
is_enabled 'table name'
describedescribe 'table name'
Dropping a Table
disable 'students'
drop 'students'
Common Filters in HBase
SingleColumnValueFilter: Filter rows based on a column’s value.
scan 'students', {FILTER => "SingleColumnValueFilter ('info', 'age', >=, 'binary:20')"}
PrefixFilter: Fetch records where the row key starts with a given prefix.
scan 'students', {FILTER => "PrefixFilter('2')"}PageFilter: Implement pagination.
scan 'students', {FILTER => "PageFilter(1)"}

🔹 e) Retrieve Name & Marks
scan 'students', {COLUMNS => ['personal:name', 'academic:marks']}
🔹 f) Employees with salary > 55000
scan 'employees', {
  FILTER => "SingleColumnValueFilter('salary','basic',>, 'binary:55000')"
}
🔹 g) Top 3 orders
scan 'orders', {FILTER => "PageFilter(3)"}
🔹 h) Students with marks > 80
scan 'students', {
  FILTER => "SingleColumnValueFilter('academic','marks',>, 'binary:80')"
}
🔹 i) Employees in IT
scan 'employees', {
  FILTER => "SingleColumnValueFilter('details','department',=, 'binary:IT')"
}
🔹 j) Orders with Credit Card
scan 'orders', {
  FILTER => "SingleColumnValueFilter('payment','mode',=, 'binary:Credit Card')"
}
🔹 k) Count rows in employees
count 'employees'
🔹 l) Oldest student
scan 'students', {
  FILTER => "SingleColumnValueFilter('personal','age',>=, 'binary:0')"
}
